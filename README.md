# 견적 제출 프로토타입 — 파트너 편

렌탈 견적 제출/수정 플로우의 AS-IS ↔ TO-BE 비교용 클릭 프로토타입 (팀 내부 논의용, 더미 데이터).

**Live:** https://delicate2330.github.io/quote-proto/

## 구성
- **1️⃣ 견적서 작성** — 4스텝 위저드 (월 렌탈료 → 고객 혜택 → 요금·특약 → 견적서 검수 그래프)
- **2️⃣ 견적 수정 (상세)** — 제품 구성 + 할인/추가 요금 카드 + 고객 혜택 + 혜택 공개
- **3️⃣ 자동견적** — 파트너센터 웹, 발송 대상 조건
- **4️⃣ 주문서 수정** — 복수 주소/계약자

## 범례
- `TO-BE · 1차 키비주얼` — 피그마 시안 재현
- `✦ 제안` — 정률 할인, 고정 태그셋, 가격 그래프 등 추가 제안
- `❓ 논의 필요` — 미결 사항

요금 카드 추가/수정/삭제 시 구간별 요금·그래프가 실시간 재계산됩니다. 단일 HTML 파일, 의존성 없음.
